Momentane Website - hoffentlich auch die entgültige im Ordner "hberf"
Zum öffnen brauchen wir php, falls ihr das nicht habt, bitte installieren
Um die Website dann im Browser zu öffnen im Terminal zuerst in hberf wechseln und dann "C:\php\php.exe" -S localhost:8000 eingeben. Danach im Browser einfach localhost:8000 eingeben.
Wir können uns alle selbst einloggen, um sich als Admin einzuloggen und mehr Rechte zu haben (Rezepte löschen, Datenbankansicht) als Benutzername BitNova und als Passwort CookingBit eingeben.
